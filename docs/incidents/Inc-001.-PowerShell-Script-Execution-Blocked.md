# Incident 0001

Title

PowerShell Script Execution Blocked

Date

2026-07-14

Severity

Low

Environment

Engineering Workstation

Problem

PowerShell refused to execute
Create-AtlasStructure.ps1

Error

running scripts is disabled...

Root Cause

Execution Policy prevents scripts.

Diagnosis

Read error message.

Verified execution policy.

Resolution

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

Verification

Script executed successfully.

Lessons Learned

Use the smallest possible scope.

Prevention

Use Process scope unless a permanent change is justified.
